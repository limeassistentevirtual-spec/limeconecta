Política de Privacidade — LIMECONECTA

Coletamos dados necessários para funcionamento do sistema, como nome, e-mail, configurações, pacientes cadastrados e valores financeiros. Os dados são usados para autenticação, organização financeira e geração de relatórios.

A usuária pode solicitar exclusão dos dados locais e encerramento do acesso. Recomendamos não cadastrar prontuários, diagnósticos ou informações clínicas sensíveis no sistema.
